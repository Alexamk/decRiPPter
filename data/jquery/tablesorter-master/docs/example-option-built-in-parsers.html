NOTE!

  • This demo is showing all of the built-in parsers (included with tablesorter's core).
  • Additional parsers are available, but must be loaded after the core.
  • Or, you have the option of writing a custom parser by using the addParser function.
  • The built-in metadata parser requires the metadata plugin; maintained for backwards compatibility. An example is not included in this demo.
